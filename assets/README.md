# Assets Directory

Place application icons here:

- `icon.png` - PNG icon (256x256 or larger) for Linux/generic use
- `icon.ico` - Windows icon file (multi-resolution)
- `icon.icns` - macOS icon file

## Creating Icons

You can use online tools like:
- https://icoconvert.com/ - Convert PNG to ICO
- https://cloudconvert.com/ - Convert between formats

Recommended icon sizes for ICO:
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256
